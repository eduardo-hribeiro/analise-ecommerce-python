# Análise Exploratória de Dados — E-commerce

Este projeto realiza uma análise exploratória de um dataset de produtos de e-commerce utilizando Python.

## Tecnologias utilizadas

- Python
- Pandas
- Matplotlib
- Seaborn

## Objetivo

Explorar características dos produtos e identificar possíveis relações entre:

- preço
- avaliações
- quantidade vendida
- marcas
- temporada

## Análises realizadas

Foram gerados os seguintes gráficos:

- Distribuição de preços (Histograma)
- Relação entre preço e avaliação (Scatterplot)
- Correlação entre variáveis numéricas (Heatmap)
- Top 10 marcas mais frequentes
- Distribuição de produtos por temporada
- Concentração de preços (KDE)
- Relação entre vendas e avaliações (Regressão)

## Insights iniciais

- A maioria dos produtos possui preço entre 50 e 200.
- Produtos com mais vendas tendem a ter avaliações ligeiramente melhores.
- A maior parte do catálogo é composta por produtos de primavera/verão.
